### BillingInfo

#### Type

data class

#### Properties

##### name

name: String

:::caution
Must match the following Regex pattern

```
Regex("^[a-zA-Z\\u0621-\\u064A-]{3,}(?:\\s[a-zA-Z\\u0621-\\u064A-]{3,})+\$")
```

:::

##### email

email: String

:::caution
Must match the following Regex pattern

```
Regex("^[a-zA-Z0-9._%+-]+@[a-zA-Z.-]+[.][a-zA-Z]{2,4}\$")
```

:::

##### phone

phone: String

:::caution
Must match the following Regex pattern

```
Regex("^\\+[0-9]{7,15}\$")
```
:::
