Promocode

  1. Navigate to Promocodes under Promocodes in the side menu.

  1. Click on Add promocode button in the top right corner.

  2. Fill in the promocode form.

Property nameValueDescriptionNotes
Namestringunique name to identify the promocode
Expiration timedate/timewhen will it expire?
Discount amountintegerfixed amount to be deducted
Discount percentageintegerpercentage of the total amount to be deducted
Amount triggerintegerMinimum amount so that the promocode takes effect
All user occurrences:integerHow many users can use this promocode?
User occurrences:integerHow many times a single user can use this promocode?
  1. click Save, you will be redirected back to Promocodes.
  1. your new promocode is ready.
