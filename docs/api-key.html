    API key

    1. Navigate to Community api keys under API Integrations in the side menu.

    1. Click on add API key button in the top right corner.
    1. Fill in the name.

    1. click Save, you will be redirected back to API keys.
    1. copy the api key value from the message that appears in the top and save it safely as it can't be retrieved later.
